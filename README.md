This is a JavaFX-based Personal Diary Manager built in NetBeans as part of the Chapter 4 Challenge. The application allows users to create, read, search, and manage diary entries with a modern, user-friendly interface.

Features
1. GUI Interface

Main Dashboard: Clean, modern JavaFX interface with navigation pane

Entry Editor: Rich text editor with formatting options (bold, italic, underline)

Entry Browser: Visual list of diary entries with previews

Search Panel: Real-time search with filters and highlighted results

Theme Support: Light and Dark mode toggle

2. Core Functionality

Write Mode: Auto-save and formatting support for diary entries

Read Mode: Beautiful display of entries with custom styling

Search Functionality: Advanced search with live results

Entry Management: Delete, rename, and organize entries visually

3. Technical Features

JavaFX Controls: Modern components for a professional look

CSS Styling: Custom stylesheets for themes and UI consistency

File I/O: Efficient file operations with progress indicators

Exception Handling: User-friendly error dialogs for smooth experience

Concurrency: Background tasks for file operations to prevent freezing

Installation

Clone the repository:

git clone https://github.com/yourusername/Chapter4_Challenge_DiaryManager_GUI.git


Open the project in NetBeans IDE

Ensure JavaFX SDK is properly configured in your project

Run the Main.java file to launch the application

Usage

Add New Entry: Navigate to Write Mode, type your diary content, and save

View Entries: Use Entry Browser to see a list of saved entries

Search Entries: Type keywords in the search panel for live results

Manage Entries: Rename, delete, or organize your diary files

Switch Themes: Toggle between Light and Dark mode from settings

Design Choices

Rich Text Editor: Chosen to enhance user experience and allow formatted diary entries

JavaFX Components: Used for modern UI, ease of styling, and flexibility

File I/O with Concurrency: Ensures smooth handling of multiple diary files without freezing the interface

Custom CSS Themes: To provide a professional and user-friendly interface
